# NRI — NRE vs NRO accounts

**Applies to:** NRI profile
**Last verified:** PENDING — not yet populated against a current source.

## What this covers
The structural distinction between NRE (foreign-sourced, exempt, no
Section 195 TDS) and NRO (India-sourced, fully taxable, bank-withheld)
accounts — flagged in the system spec as the single most common NRI
filing error to guard against (treating NRO income as exempt because
"it's an NRI account").
