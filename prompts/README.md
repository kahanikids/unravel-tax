# Prompt Pack

Start with `00-master-guide.md`.

Copy the whole file into a new AI chat. It is the only prompt a first-time
user should need at the start: it handles orientation, gives a personal
document checklist, and switches to the extraction and explanation prompts
only when those steps are needed.

The two supporting prompts are included for maintainers and for users who
need to restart a specific step:

- `01-extract-statement.md` extracts one document into clean transaction
  rows.
- `02-explain-my-results.md` explains completed results in plain language.

Do not ask users to choose between these prompts up front. The master
guide should remain the single obvious entry point.
